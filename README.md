# macOS Toolbox

A lightweight, interactive terminal interface for managing Homebrew packages.
<img width="924" height="462" alt="image" src="https://github.com/user-attachments/assets/23b8ffdf-71fb-40c4-8690-db612a0d03f6" />

## Features

* Interactive Search
* Installation Queue
* Uninstall Mode
* Upgrade Prompt
* Automatic Cleanup

## Navigation & Controls

| Key | Action |
| :--- | :--- |
| TAB | Search in Homebrew |
| V | Cycle views |
| SPACE | Select or deselect |
| ENTER | Finalize and run |
| Arrows | Navigate lists |

## Requirements

* ~~A Mac with Homebrew installed.~~
  The script will install Homebrew if it is not already installed.
* Zsh (the default shell on macOS).
